Pré-requis

Vous avez à disposition une application Ruby on Rails. Nous ferons nos exercices sur celle-ci.

Cette application contient un certain nombre de fonctionnalités déjà mises en place.

Yarn & Webpack

Le frontend est un univers qui évolue très vite. Aujourd'hui, Webpack est l'outil frontend defacto lorsqu'il s'agit de packager son Javascript.

Il est temps de s'acclimater à ce nouvel environnement.

Exercice

L'application fournie vient avec le framework Bootstrap mais le javascript n'est pas installé. Nous allons corriger cela.

  • Prenez le temps de découvrir chaque fichier / dossier apporté par webpack ou webpacker
  • Ajoutez le package jquery
  • Ajoutez le package bootstrap ainsi que popper.js
  • Configurez webpack pour que jQuery soit accessible de base dans vos packs
  • Importez bootstrap dans votre pack

Un peu d'aide

Configuration de jQuery :

// config/webpack/environment.js
const webpack = require('webpack');

environment.plugins.prepend('Provide',
  new webpack.ProvidePlugin({
    $: 'jquery',
    jQuery: 'jquery'
  })
)

Import de bootstrap dans votre pack :

// app/javascript/packs/application.js
import "bootstrap";

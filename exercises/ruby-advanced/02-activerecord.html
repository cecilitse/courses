Pré-requis

Vous avez à disposition une application Ruby on Rails. Nous ferons nos exercises sur celle-ci.

Cette application contient un certain nombre de fonctionnalités déjà mises en place.

Index

Le 1er objectif de ce module sera la mise en place d'index.

Exercice

  • Découvrez les fonctionnalités en place
  • Découvrez le schéma de données
  • Analysez les requêtes
  • Déterminez où mettre des index
  • Mettez-les en place

Requêtes

Après avoir posé de l'index, passons à l'optimisation des requêtes existantes.

Exercice

  • Analysez à nouveau les fonctionnalités en place
  • Déterminez comment les optimiser
  • Mettez en place toute optimisation que vous jugez necessaire

Alléger les contrôleurs

Certains de nos contrôleurs se retrouvent très chargés. D'autres se retrouvent avec des requêtes dupliquées. Il est temps de nettoyer tout cela.

Exercice

Allégez les contrôleurs en extrayant certaines requêtes dans des scopes ou Query Objects.

Éditeur de texte

1er objectif de la journée : s'assurer que l'éditeur de texte est correctement configuré.

Exercice

  • Assurez vous que votre éditeur est configuré sur 2 espaces lorsque l'on utilise la touche Tab.
  • Lorsque vous sauvegardez un fichier, votre éditeur devrait s'assurer que vous avez toujours une ligne vide en fin de fichier.

Rubocop

Rubocop est votre compagnon vers un code de bonne qualité.

Exercice

  • Choisissez un de vos projets
  • Installez-y Rubocop
  • Choisissez un fichier à tester.
    Veillez à ce que le fichier soit un minimum conséquent.
  • Lancez rubocop sur ce fichier
  • Analysez le résultat
  • Configurez un minimum Rubocop : retirez tout ce qui ne vous semble pas pertinent.
  • Lancez à nouveau Rubocop
  • Corrigez votre fichier jusqu'à ne plus avoir d'offenses

Rubycritic

Maintenant que nous avons rubocop qui nous aide au niveau macro, il est temps de prendre un peu de recul. Notre nouvel objectif est de voir l'état global de notre base de code.

Exercice

  • Toujours sur le même projet
  • Installez-y Rubycritic
  • Déterminez où se trouve le plus gros de votre logique métier.
    Listez tous les dossiers concernés.
    • Dans tout le dossier app/ ?
    • Dans uniquement quelques dossiers dans app/ ?
    • Dans le dossier lib/ également ?
    • Ailleurs ?
  • Lancez Rubycritic sur les dossiers sélectionnés.
  • Analisez le résultat et déterminez les zones à traiter en priorité.

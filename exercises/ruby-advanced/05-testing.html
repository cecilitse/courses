Pré-requis

Vous avez à disposition une application Ruby on Rails. Nous ferons nos exercices sur celle-ci.

Cette application contient un certain nombre de fonctionnalités déjà mises en place.

RSpec

Il nous faut d'abord installer notre framework de test.

Exercice

  • Installer et configurer RSpec
  • Installer factory_bot
  • Le configurer à minima pour le modèle User

Tests unitaires

1er objectif, la mise en place de tests unitaires sur notre application existante.

Exercice

  • Mettre en place au moins un test unitaire de modèle (méthode de classe ou d'instance)
  • Mettre en place au moins un test unitaire de scope ou query object
  • Mettre en place au moins un test unitaire de service

Tests système

Nous passons ensuite sur du test sytème. L'objectif est de simuler le parcours utilisateur.

Exercice

  • Configurer RSpec pour les tests sytèmes
  • Mettre en place au moins un test sytème sur une des fonctionnalités existantes.

Test Driven Development (TDD)

Jusqu'ici, nous avons mis en place des tests sur des fonctionnalités existantes. Passons au niveau supérieur : implémentons une nouvelle fonctionnalité.

Exercice

  • Écrivez le test de votre nouvelle fonctionnalité
  • Lancez votre test, regardez le planter (RED)
  • Implémentez votre fonctionnalité
  • Lancez votre test, regardez le retourner un succès (GREEN)
  • Améliorez votre code (REFACTOR)
  • Lancez votre test, vérifier que cela fonctionne toujours comme attendu

Metrics

Regardons un peu où nous en sommes concernant nos tests.

Exercice

  • Installez Simplecov
  • Configurez-le
  • Pensez à créer des groupes pour vos nouveaux composants
  • Lancez à nouveau vos tests
  • Analysez la couverture de code

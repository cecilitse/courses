Pré-requis

Vous avez à disposition une application Ruby on Rails. Nous ferons nos exercises sur celle-ci.

Cette application contient un certain nombre de fonctionnalités déjà mises en place.

Elastisearch

Nous y voilà, il est temps d'installer notre 1er moteur de recherche.

Exercice

  • Installez le paquet default-jre via aptitude
  • Installez Elasticsearch
  • Installez Searchkick

Recherche fulltext

L'objectif de ce module est d'ajouter une fonctionnalité de rercherche fulltext sur le listing que nous avons actuellement sur notre application.

Exercice

  • Déterminez quel modèle indexer
  • Déterminez les propriétés du document qui sera indexé
  • Mettez en place l'index
  • Mettez en place la rercherche fulltext

Aggrégats

Terminons par la mise en place de filtres intelligents, les aggrégats.

Exercice

  • Déterminez quels pourraient être les filtres
  • Au besoin, modifiez votre index
  • Mettez en place ces filtres sur la page de listing
  • Veillez à ce que les filtres fonctionnent bien avec la recherche fulltext
